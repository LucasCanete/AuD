# Inhalt 
Beispiel eines Sortier-Algorithmus (*Selection Sort*) als Java-Projekt.

# Verwendung
Das Projekt dient der Demonstration und setzt außer einem Java JDK keine 
keine weitere Software voraus. Es kann mit unterschiedlichen IDEs geöffnet werden (getestet
wurden VS Code, IntelliJ und Netbeans).

Nützliche Klassen zur Verwaltung von Feldern können natürlich auch für eigene Projekte 
verwendet werden.

Die Hauptklasse ist `Call_Wrapper`.
